# Web 3<br>
## Web 3是什麼?<br>
是全球資訊網發展的一種概念，提倡去中心化也就是資料不集中於任何伺服器中，區塊鏈使之得以真正實現。區塊鏈中資料無需集中在政府或企業管理且能永久保存在網路上的技術<br>
